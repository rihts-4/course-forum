# Course Forum

A peer-to-peer course forum for university students in a single department. This platform allows students to create and engage in discussions about their courses, ask questions, and connect with peers and professors.

This project is currently in the early stages of development. The features listed below are planned for implementation.

## Features

*   **Course-Specific Threads:** Create discussion threads for each course in the department.
*   **User Authentication:** Secure sign-up and login for students.
*   **User Profiles:** Basic profiles showing user's name, courses, and contributions.
*   **Q&A and Discussions:** Post questions, answers, and general discussion topics.
*   **Voting System:** Upvote and downvote posts and comments to highlight valuable content.
*   **Search Functionality:** Easily search for specific topics or keywords across all courses.
*   **Admin Functionalities:** Tools for administrators to manage users, courses, and site-wide settings.
*   **Review Moderation:** A system for flagging and reviewing content to maintain community guidelines.
*   **Professor-Tagged Threads:** Users can create threads and tag specific professors, allowing for focused discussions and questions directed at faculty.
*   **AI Chatbot Assistant:** An interactive chatbot that helps users find relevant discussions and get recommendations based on their profile and queries.
*   **Course/Professor Rating System:** A comprehensive system for students to rate and review courses and professors.
